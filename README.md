## Getting Started

### Prerequisites

Need to have installed before they can use your project.

- Node.js (20.11.1)
- yarn
- mui (Material UI)
### Installation

1. Clone the repository:

```bash
git clone https://github.com/dokdo-sh/Dokdo.sh-FE.git
```

2. Install packages
```
yarn install
```
3. Run dev server
```
yarn start
```
